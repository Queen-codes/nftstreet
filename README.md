## Getting Started

- This project was created with [Vite](https://vitejs.dev/) which requires [Node.js](https://nodejs.org/) v14.18+, v16+ to run.

- Install the dependencies and devDependencies and start the server.

```sh
cd nftstreet
npm install
npm run dev
```
